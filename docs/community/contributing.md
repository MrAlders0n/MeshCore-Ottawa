
# How to Request Changes to OttawaMesh.ca

If you want to add a new repeater, update deployment details, or fix documentation, the easiest way is to open a **GitHub Issue**.

You do not need to know Git, GitHub, or how to edit the website. Just follow the steps below.

---

## 1. Go to the Issues page

Visit the Ottawa Mesh GitHub repository and click **Issues**.

Link: <https://github.com/MrAlders0n/MeshCore-NCR/issues>

---

## 2. Click "New Issue"

Choose the template that matches what you want to do:

- **Repeater Deployment Update**
- **New Documentation Request**
- **Documentation Correction**
- **Incorrect Information**

---

## 3. Fill out the form

Provide as much detail as possible. 

---

## 4. Submit the issue

A maintainer will:

1. Review the request  
2. Make the change  
3. Close the issue when complete  

If we need more information, we’ll comment on the ticket.

---

## 5. Optional — Advanced Contributions

If you know GitHub and want to edit the markdown files directly:

1. Fork the repo  
2. Edit the relevant file in `docs/deployment/`  
3. Open a pull request  

---

## Thank You

OttawaMesh.ca is a community-maintained resource, and contributions from all regions help keep the network growing.
